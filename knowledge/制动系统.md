# 制动系统维修指南

## 常见故障现象

### 刹车异响
- **可能原因：**
  - 刹车片磨损到报警片 → 尖利金属摩擦声
  - 刹车片材质过硬/含金属颗粒 → 刹车时有摩擦声
  - 刹车盘变形 → 刹车时方向盘抖动
  - 刹车卡钳导向销缺油/卡滞 → 单侧刹车片偏磨
  - 制动背板与刹车盘接触 → 低速时"沙沙"声
- **诊断步骤：**
  1. 拆检刹车片厚度（≤3mm需更换）
  2. 检查刹车盘表面是否有深沟或裂纹
  3. 检查刹车盘跳动量（≤0.05mm）
  4. 检查导向销是否灵活
  5. 听声音来源：前轮还是后轮、制动时还是行驶时

### 刹车软（踏板行程过大）
- **可能原因：** 制动管路有空气、制动液泄漏、制动主缸内漏、制动液含水量过高
- **诊断步骤：**
  1. 目视检查制动液储液罐液位
  2. 检查制动管路有无渗漏（尤其轮缸处）
  3. 踩刹车踏板3次后脚感变硬→系统内有空气（需排气）
  4. 踏板缓慢下沉→主缸内漏
  5. 测试制动液含水量（>3%需更换）

### 刹车跑偏
- **可能原因：** 单侧刹车分泵卡滞、刹车油管堵塞、轮胎气压不等、悬挂系统磨损
- **诊断步骤：**
  1. 检查左右轮胎气压和磨损情况
  2. 路试注意跑偏方向（向右跑偏→左前轮或右后轮制动力不足）
  3. 用红外测温仪检测制动后各轮毂温度（温差>30℃说明异常）
  4. 检查制动分泵回位是否正常

### ABS/EBD 故障灯亮
- **可能原因：** 轮速传感器脏污或损坏、ABS齿圈破损、ABS泵故障、保险丝熔断
- **诊断步骤：**
  1. 读取ABS系统故障码
  2. 检查轮速传感器间隙（正常0.3-1.2mm）和清洁度
  3. 用示波器检测轮速传感器波形
  4. 检查ABS齿圈是否有缺齿或裂纹
  5. 测量ABS泵电机电阻

## 制动液规格与更换
- DOT3：干沸点205℃，吸水性强，经济选择
- DOT4：干沸点230℃，家用车标配
- DOT5.1：干沸点260℃，高性能车
- **严禁混用不同类型制动液**
- 更换周期：2年或4万公里（先到为准）
- 更换方法：四轮顺序放油排气（从最远端到最近端：右后→左后→右前→左前）

## 制动盘与刹车片
- 刹车片最小厚度：含背板总厚度≥7mm，摩擦材料≥3mm
- 刹车盘最小厚度：盘面标注有"MIN TH"字样
- 新刹车片/盘磨合：初始200公里避免急刹车
