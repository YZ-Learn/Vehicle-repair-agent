# 底盘与悬挂系统维修指南

## 常见故障现象

### 方向盘抖动
- **行驶时抖动：**
  - 60-80km/h→轮胎动平衡不良
  - 80-120km/h→车轮定位失准或轮胎偏磨
  - 刹车时抖动→刹车盘变形
  - 加速时抖动→半轴或传动轴磨损
- **诊断步骤：**
  1. 做轮胎动平衡（注意检查配重块是否脱落）
  2. 检查轮胎胎面磨损是否均匀（不均匀→做四轮定位）
  3. 检查转向拉杆球头和下摆臂衬套间隙
  4. 检查传动轴十字节和内球笼
  5. 路试区分振动来源（方向盘、座椅、地板）

### 跑偏
- **可能原因：** 四轮定位参数偏差、轮胎气压不一致、悬挂部件变形/磨损、不同花纹/磨损程度的轮胎混用
- **诊断步骤：**
  1. 检查四轮胎压
  2. 左右换胎测试（如果跑偏方向变化→轮胎问题）
  3. 检查悬挂部件（下摆臂、稳定杆连杆、减震器）
  4. 做四轮定位，调整前束/外倾角

### 底盘异响
- **过减速带"咯吱"响：** 稳定杆胶套老化、下摆臂衬套磨损、减震器顶胶老化
- **低速转向"咔咔"响：** 外球笼磨损（前驱车常见）
- **烂路"当当"响：** 减震器失效、悬挂螺栓松动、平衡杆连杆断裂
- **金属碰撞声：** 排气管碰底盘、制动挡板碰刹车盘
- **诊断方法：** 用听诊器/长螺丝刀接触各部件听音定位，上下晃动轮胎检查间隙

### 车身侧倾过大
- **可能原因：** 减震器失效、稳定杆连杆断裂/松旷、弹簧疲劳、稳定杆衬套磨损
- **诊断步骤：**
  1. 按压车身四角检查回弹（应迅速回弹不振荡）
  2. 检查减震器是否漏油
  3. 检查稳定杆连杆球头间隙
  4. 举升检查稳定杆衬套

## 转向系统

### 电动助力转向（EPS）
- **故障灯亮/助力消失：** 扭矩传感器故障、EPS控制单元故障、电动机故障、供电异常
- **转向沉重：** 轮胎气压不足、EPS/ECU设置问题、EPS减速机构卡滞
- **转向回正不良：** 主销后倾角不对、转向柱卡滞
- **诊断：** 读取EPS故障码+数据流（扭矩传感器信号、电机电流、车速信号）

### 液压助力转向（HPS）
- **转向沉重：** 助力泵皮带打滑、助力油液位低、转向机内部泄漏
- **转向异响（液压啸叫）：** 助力油不足、油路有空气、助力泵磨损
- **漏油：** 高压管接头、转向机防尘套内漏
- **检查方法：**
  1. 检查助力油液位和颜色（红色→正常，黑色/乳化→更换）
  2. 打死方向检查有无异响
  3. 测量助力泵压力
  4. 检查转向机防尘套是否破裂鼓包

## 轮胎与轮毂
- **胎压监测系统（TPMS）：**
  - 直接式：轮内传感器，实时检测气压/温度
  - 间接式：通过ABS轮速传感器检测转速差（漏气轮转速略快）
  - 更换轮胎后需执行TPMS复位/学习
- **轮胎磨损模式诊断：**
  - 两侧磨损→胎压不足
  - 中间磨损→胎压过高
  - 单侧磨损→外倾角偏差
  - 羽毛状磨损→前束角偏差

## 四轮定位参数

| 参数 | 含义 | 常见调整方法 |
|------|------|-------------|
| 前束（Toe） | 轮胎前端向内/外偏 | 调整转向拉杆长度 |
| 外倾角（Camber） | 轮胎上方向内/外倾斜 | 调整下摆臂偏心螺栓/塔顶螺丝 |
| 主销后倾角（Caster） | 转向轴线前后倾斜 | 调整下摆臂位置（部分车型不可调） |
| 推进角（Thrust Angle） | 后轴推进方向 | 调整后扭力梁/后下摆臂 |

- 建议检测频率：每2万公里或更换轮胎后
- 更换悬挂部件后必须重做定位
