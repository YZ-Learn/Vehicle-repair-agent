# 电气系统维修指南

## 蓄电池
### 检测与维护
- **静态电压：** ≥12.4V（低于12.0V需充电或更换）
- **启动电压：** 启动时降至≥10.0V（低于9.6V说明蓄电池老化）
- **内阻：** 免维护蓄电池≤6mΩ，AGM蓄电池≤4mΩ
- **SAE标准：** CCA值≥原车标注值的70%
- **蓄电池桩头：** 白色或蓝色结晶物→用开水冲洗后涂凡士林防腐蚀

### 亏电排查
1. 测量静态放电电流（正常20-50mA，超过80mA异常）
2. 逐个拔保险丝定位漏电电路
3. 常见漏电源：加装的记录仪/导航/功放、门锁模块不休眠、发电机整流二极管击穿

## 发电机
### 常见故障
- **不发电：** 调节器损坏、电刷磨损、转子断路、整流器击穿
- **发电量不足：** 皮带打滑、定子绕组局部短路、轴承磨损
- **异响：** 轴承缺油、转子扫膛

### 检测方法
1. 静态：测量各端子电阻（转子2-5Ω，定子三相导通）
2. 动态：启动后测量B+端子电压（13.5-14.5V正常）
3. 加载测试：开大灯+鼓风机+后窗加热，电压应≥13.0V
4. 检查发电机皮带张紧度（按压皮带中部下陷8-12mm）

## 起动机
### 常见故障
- **不转：** 电磁开关故障、电刷磨损、定子线圈断路
- **空转：** 单向离合器打滑
- **异响：** 齿轮磨损、安装螺栓松动

### 检测方法
1. 打开点火开关到启动档，测量电磁开关S端子电压（≥12V）
2. 短接B+和S端子测试起动机是否运转
3. 拆下检查电刷长度（≥新品的1/2）
4. 检查飞轮齿圈是否磨损

## 灯光系统
### 常见问题
- **灯泡频繁烧坏：** 发电机电压过高、灯座接触不良、劣质灯泡
- **LED灯闪烁：** 解码电阻/电容故障、CAN总线匹配问题
- **自适应大灯故障：** 水平传感器故障、步进电机卡滞

## 传感器供电与搭铁
- 传感器供电通常为5V参考电压（ECU提供）
- 检查搭铁点：散热器框架附近、发动机缸盖后部、左右翼子板内
- 搭铁不良导致的现象：传感器信号飘移、执行器工作不稳定、仪表盘指针抖动

## CAN 总线诊断
- CAN H 电压：2.5-3.5V
- CAN L 电压：1.5-2.5V
- 正常差值：>2V（唤醒后）
- 终端电阻：120Ω（在两端ECU内），总线总阻值约60Ω
