# 发动机系统维修指南

## 常见故障现象与诊断

### 发动机无法启动
- **可能原因：** 蓄电池电压不足、起动机故障、燃油泵不工作、点火系统故障、发动机控制单元（ECU）供电异常
- **诊断步骤：**
  1. 检查蓄电池电压（≥12.4V）
  2. 听起动机是否运转（不转→检查起动机继电器和保险丝）
  3. 检查燃油泵是否有工作声（打开点火开关时应有2秒嗡嗡声）
  4. 检查点火系统（拔下一缸高压线接备用火花塞，检查是否有火）
  5. 读取故障码

### 怠速不稳
- **可能原因：** 节气门积碳、怠速控制阀故障、真空泄漏、喷油器堵塞、进气流量传感器（MAF）污染
- **诊断步骤：**
  1. 读取数据流，观察怠速转速和目标值偏差
  2. 检查节气门位置传感器（TPS）信号
  3. 用化清剂在进气管路喷射，检测转速变化判断真空泄漏
  4. 清洗节气门和怠速控制阀
  5. 检查MAF传感器数值（正常：2-5g/s @怠速）

### 加速无力
- **可能原因：** 燃油压力不足、空气滤清器堵塞、三元催化器堵塞、涡轮增压器故障（增压车型）、点火正时偏差
- **诊断步骤：**
  1. 路试读取数据流，检查加速时进气量和燃油修正值
  2. 测量燃油压力（怠速：2.5-3.5bar，急加速时保持稳定）
  3. 检查空气滤清器
  4. 检查增压管路是否泄漏（增压车型）
  5. 监测前后氧传感器信号

### 发动机过热
- **可能原因：** 冷却液不足、节温器卡滞、水泵故障、散热器堵塞、电子风扇不工作
- **诊断步骤：**
  1. 检查冷却液液位和颜色
  2. 检查电子风扇是否运转（水温到达风扇开启温度时）
  3. 触摸上下水管温差判断节温器是否正常（上热下冷→节温器未打开）
  4. 检查水泵皮带和有无泄漏
  5. 检查散热器表面是否堵塞

### 机油消耗过大
- **可能原因：** 活塞环磨损、气门油封老化、涡轮增压器油封泄漏、曲轴箱通风系统（PCV）故障
- **区分排蓝烟还是渗漏：**
  - 冷启动冒蓝烟→气门油封
  - 加速冒蓝烟→活塞环
  - 怠速冒蓝烟→涡轮增压器

## 传感器与执行器

| 传感器 | 常见故障 | 检测方法 |
|--------|---------|---------|
| 曲轴位置传感器（CKP） | 信号丢失→无法启动 | 用示波器检测波形，电阻200-1000Ω |
| 凸轮轴位置传感器（CMP） | 信号偏差→启动延迟 | 检查安装间隙，正常0.5-1.0mm |
| 氧传感器 | 响应迟缓→油耗高 | 读取信号电压，正常0.1-0.9V波动 |
| 爆震传感器 | 不检测爆震→点火提前角受限 | 用震动台检测输出信号 |
| 节气门位置传感器 | 信号跳变→加速顿挫 | 测量电阻，应随开度线性变化 |

## 正时系统
- 正时皮带更换周期：6-10万公里（具体看车型手册）
- 正时链条：一般免维护，但张紧器可能失效
- 正时对不准的症状：怠速抖动、加速无力、严重时活塞顶气门
